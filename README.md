# House Price Prediction using Linear Regression

## Objective

Build a Linear Regression model to predict house prices based on area, number of bedrooms and number of bathrooms.

## Dataset

Housing.csv

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Steps Performed

1. Loaded the dataset.
2. Explored the dataset.
3. Checked for missing values.
4. Selected input features and target variable.
5. Split the dataset into training and testing sets.
6. Trained a Linear Regression model.
7. Predicted house prices.
8. Evaluated the model.
9. Visualized Actual vs Predicted prices.
10. Performed a custom prediction.

## Results

R2 Score: 0.4559

## Conclusion

Successfully implemented a Linear Regression model to predict house prices.
